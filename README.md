# weblog_metric
This is my graduation project, It based on Django.

## Contents
    1. Hadoop+Hive构建数据仓库
    2. django+echarts网站开发
    3. 数据清洗，数据模型构建

## Quick Start

#### test
    python manage.py test

#### makemigrations
    python manage.py makemigrations
    
#### migrate
    python manage.py migrate
    
#### run
    python manage.py runserver
